# Controle Financeiro Pessoal

Sistema em Python para registrar gastos e dívidas, calcular total restante e número de parcelas.

## Estrutura do projeto

- `src/` → arquivos .py do projeto
- `data/` → CSVs com os dados de gastos e dívidas

## Como usar

1. Abra o terminal na pasta `src/`
2. Rode: `python main.py`
3. Siga o menu para registrar ou listar gastos e dívidas.
